CLI Tool

Manage your domain registrant contacts at scale

A powerful command-line interface for updating Cloudflare domain registrant contact information across multiple domains. Templates, bulk operations, and interactive workflows.

$ cloudflare-registrar login
$ cloudflare-registrar list
$ cloudflare-registrar template save personal
$ cloudflare-registrar bulk-update --template personal
Capabilities

Everything you need to manage registrant data

☰

Domain Listing

View all domains with current registrant info in table or JSON format. Filter by pattern with regex support.

Interactive Updates

Update a single domain's registrant contact interactively with current values prefilled for easy editing.

Getting started

Up and running in two minutes

1
Prerequisites — Node.js ≥ 14.0.0 and npm or yarn installed.
2
Install dependencies — Clone the repo and run npm install.
3
Link globally — Run npm link to make the CLI available everywhere.
4
Authenticate — Run cloudflare-registrar login to set up your API token.
Terminal
1$ git clone https://github.com/shishirsharma/cloudflare-registrar.git
2$ cd cloudflare-registrar
3$ npm install
4$ npm link
5
6# Authenticate with Cloudflare
7$ cloudflare-registrar login
Workflow

Four steps to updated contacts

1.
cloudflare-registrar login

Authenticate with Cloudflare. Opens your browser, guides you through token creation, validates and saves securely.

2.
cloudflare-registrar list

List all your domains with current registrant details. Use --filter to narrow results with regex patterns.

3.
cloudflare-registrar template save personal

Create a reusable contact template. The CLI prompts you for each field interactively.

4.
cloudflare-registrar update example.com --template personal

Apply a template to a domain. Previews changes and asks for confirmation before writing.

Reference

Complete command reference

login
Interactive authentication with Cloudflare. Opens browser, guides token creation, validates and saves.
alias: auth
config
Manage API configuration. Initialize, view, or reset your stored credentials.
init show reset
list
List all domains with registrant information. Supports table and JSON output.
--json --filter <pattern>
update
Update a single domain's registrant contact interactively or from a template.
--template <name> --interactive --json
bulk-update
Bulk update multiple domains from a file. Supports dry-run to preview changes.
--domains <file> --template <name> --dry-run --json
template
Manage reusable contact templates. Create, list, view, or delete saved profiles.
save <name> list load <name> delete <name>
